ChestX.ai

An integrated and intelligent system to empower radiologists.

Diagnose with Confidence

ChestX.ai provides diagnosis assistance to radiologists by integrating chest X-ray images with patient's electronic medical records, and predicting multiple chest diseases using state-of-art Convolutional Neural Network algorithms.

Architecture

Patientsâ€™ chest X-ray images are uploaded and stored on IBM Cloud HIPAA-compliant object storage. The Radiologist Review Portal is built in PowerApps, of which the backend database is hosted on Azure SQL server.

Data

We trained on more than 110,000 X-ray images from 30,805 unique patients with 14 different labeled diseases from NIH dataset as well as 266 images from newly aggregated COVID-19 chest X-ray dataset.

Approach

We employ a two-tier approach towards X-ray classification. At Tier 1, a model determines whether the patient is diseased or healthy. If the model predicts that the patient is not normal, Tier 2 is triggered to begin multi-class disease classification as well as COVID-19 prediction.

Model

The three models developed for ChestX.ai employ state-of-the-art DenseNet-121 convolutional neural network (CNN) model architecture. Additionally, heatmaps for localized important regions are generated using Gradient-weighted Class Activation Mapping (Grad-CAM).
